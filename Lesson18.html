Циклы: for, while, do-while

Циклы позволяют выполнять повторяющиеся действия без необходимости писать один и тот же код снова и снова. В
JavaScript есть три основных вида циклов: for, while и do-while

Цикл for:

Цикл for используется, когда заранее известно количество повторений

Цикл while:

Цикл while используется, когда количество повторений неизвестно заранее, но есть условие, при котором цикл должен продолжаться

JavaScript

В этом примере мы начинаем счет с нуля, на каждом повторении добавляем 1 и код будет выполняться, пока i < 5

Цикл do-while:

Цикл do-while похож на while, но проверка условия происходит после выполнения тела цикла, поэтому код внутри цикла всегда выполняется хотя бы один раз

JavaScript

Циклы for, while и do-while предоставляют различные способы автоматизации повторяющихся действий в вашем коде, что делает вашу программу более эффективной и удобной для использования. Давайте потренеруемся
