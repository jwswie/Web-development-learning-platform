Основы CSS

CSS (Cascading Style Sheets) - это способ придания стиля вашему веб-сайту. С помощью CSS вы можете изменить цвета, шрифты, размеры и расположение элементов на вашей странице

Внутренние и внешние стили:

CSS можно применять к HTML документу двумя способами: внутренние и внешние стили. Внутренние стили помещаются
внутри тега <style> внутри вашего HTML документа, а внешние стили хранятся в отдельных CSS файлах и подключаются к вашему HTML документу с помощью тега <link>

Селекторы, свойства и значения:

Селекторы используются для выбора элементов HTML, к которым вы хотите применить стили. Свойства определяют
конкретные аспекты стиля, такие как цвет или размер текста, а значения определяют, какими будут эти аспекты
