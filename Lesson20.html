Объекты и массивы: работа с данными и методами объектов и массивов

В JavaScript объекты и массивы - полезные инструменты для хранения и организации данных. Они помогают
структурировать информацию так, чтобы её можно было легко использовать в вашем коде

Массивы:

Массив - это набор элементов, к которым можно обращаться по их порядковому номеру, который называется индексом

JavaScript

Объекты:

Объекты - это коллекции пар ключ-значение. Они позволяют организовать данные в структурированном формате

JavaScript

Работа с данными и методы:

Массивы и объекты имеют различные методы, которые позволяют выполнять различные операции над данными. Например, метод push() для массивов добавляет новый элемент в конец массива, а метод Object.keys() для объектов возвращает массив ключей объекта

JavaScript

Понимание работы с массивами и объектами поможет вам эффективно организовывать и использовать данные в ваших программах. Давайте потренеруемся

Задание на закрепление материала:

Создайте простую веб-страницу. Ваша страница должна содержать следующий код:

  • Создайте пустой массив tasks
  • Создайте функцию addTask, которая принимает строку task и добавляет её в конец массива tasks
  • Создайте функцию removeTask, которая принимает индекс элемента в массиве и удаляет соответствующую задачу из массива tasks
  • Создайте функцию listTasks, которая выводит список всех задач в массиве tasks
