Изменение Содержимого и Стилей HTML-элементов

DOM (Document Object Model) - это способ, с помощью которого браузеры представляют веб-страницы в виде структуры объектов, с которыми JavaScript может взаимодействовать. Манипуляция DOM позволяет нам изменять содержимое и стили HTML-элементов на веб-странице

Изменение Содержимого:

Мы можем изменять содержимое HTML-элементов, например, текст или атрибуты
