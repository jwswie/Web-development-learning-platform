Операторы и условные конструкции: if, else, switch

Операторы if, else и switch - это инструменты в JavaScript, которые позволяют вашему коду принимать разные решения на основе условий

Оператор if:

Оператор if позволяет выполнять код, если указанное условие истинно

JavaScript

В этом примере, если переменная number делится на 2 без остатка, то будет выведено сообщение "Число четное"

Оператор else:

Оператор else позволяет выполнить код, если условие в операторе if ложно

JavaScript

Оператор switch:

Оператор switch позволяет проверить разные варианты значения одной переменной

JavaScript

Операторы и условные конструкции позволяют вашему коду делать различные действия в зависимости от ситуации, что
делает его более гибким. Давайте потренеруемся

Задание на закрепление материала:

Создайте простую веб-страницу. Ваша страница должна содержать следующий код:

  • Создайте переменные для названия, цены и категории товара
  • Определите скидку на товар в зависимости от его категории. Например, если категория товара "Еда", то скидка 10%
  • Рассчитайте стоимость товара со скидкой
  • Выведите название, итоговую стоимость и есть ли на товар скидка в alert
