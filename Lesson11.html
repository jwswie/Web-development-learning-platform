Текстовое оформление: размер, цвет, выравнивание, шрифты

Когда вы создаёте веб-сайт, важно сделать его привлекательным и легко читаемым для посетителей. Один из способов
сделать это - правильно оформить текст на странице

Изменение размера текста:

Вы можете легко изменить размер текста, используя свойство font-size в CSS

CSS

Изменение цвета текста:

Чтобы изменить цвет текста, используйте свойство color

CSS

Выравнивание текста:

Чтобы выровнять текст по центру, слева или справа, используйте свойство text-align

CSS

Изменение шрифта:

Чтобы изменить шрифт текста, вы можете использовать свойство font-family

CSS

Это простые примеры того, как вы можете оформить текст на вашем веб-сайте с помощью CSS. Давайте потренеруемся

Задание на закрепление материала:

Создайте веб-страницу с текстом. Добавьте на неё такие стили:

  • Установите размер шрифта для всех параграфов на 16 пикселей
  • Установите цвет текста для заголовка на синий (#0000FF)
  • Установите выравнивание текста для всех параграфов по центру
  • Установите шрифт для всего текста на ваш выбор
