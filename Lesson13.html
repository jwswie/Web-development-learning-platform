Flexbox и Grid: методы создания адаптивных макетов

При создании веб-страницы важно, чтобы она хорошо выглядела на любом устройстве, будь то компьютер, планшет или смартфон. Для этого существуют два инструмента: Flexbox и Grid. Они позволяют нам управлять расположением элементов
на странице и создавать адаптивные макеты, которые легко адаптируются к различным размерам экранов

Flexbox:

Это метод компоновки элементов в строку или столбец. Он идеально подходит для создания сложных макетов с различными типами содержимого

HTML

CSS

  • display: flex;: Определяем контейнер как flex-контейнер
  • justify-content: space-between;: Распределяем элементы по всей ширине контейнера с равными промежутками между
    ними
  • flex: 1;: Элементы будут занимать равное пространство внутри контейнера

Grid:

Это двумерная система разметки, позволяющая создавать сложные сетки с ячейками

HTML

CSS

  • display: grid;: Определяем контейнер как сетку
  • grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));: Создаём колонки, которые автоматически масштабируются в зависимости от размера экрана
  • gap: 20px;: Промежуток между элементами

Flexbox и Grid - это очень удобные инструменты для создания адаптивных макетов на веб-страницах. Давайте потренеруемся

Задание на закрепление материала:

Создайте простую веб-страницу. Добавьте на неё такие стили:

  • Создайте на странице контейнеры с блоками контента
  • Используйте Flexbox для организации блоков внутри одного из контейнеров
  • Используйте Grid для создания сетки с блоками контента в другом контейнере
  • Стилизуйте блоки контента с помощью CSS (по желанию)
