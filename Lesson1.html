Введение в HTML

HTML (HyperText Markup Language) - язык разметки, который используется для создания веб-страниц. Когда мы открываем
веб-страницу в браузере, браузер интерпретирует HTML-код и отображает содержимое в виде веб-страницы

Структура документа HTML:

HTML-документ состоит из различных элементов, которые определяют структуру и содержимое страницы. Основной
элемент называется <html>, и он содержит все остальные элементы на странице. Внутри <html> есть два основных блока: <head> и <body>

  • <head>: Этот блок содержит информацию о документе, такую как заголовок страницы (<title>), подключение стилей
    CSS, метаданные и скрипты
  • <body>: Здесь располагается основное содержимое веб-страницы, такое как текст, изображения, ссылки и другие
    элементы

Теги и элементы:

В HTML мы используем теги для определения различных типов содержимого на странице. Теги обрамляют элементы и дают
им определённое значение или функцию

HTML

В данном примере:

  • <h1>: Тег для заголовка верхнего уровня, который обозначает самый важный заголовок на странице
  • <p>: Тег для абзаца текста
  • <img>: Тег для вставки изображения. src указывает на путь к изображению, а alt - на альтернативный текст, если
    картинка не прогрузится
  • <a>: Тег для создания ссылок. href указывает на адрес, на который происходит переход при клике на ссылку

HTML - это основа веб-разработки, и понимание его основ является важным шагом для создания красивых и
функциональных веб-страниц. Давайте потренеруемся

Задание на закрепление материала:

Создайте веб-страницу с информацией о себе. Ваша страница должна содержать следующие элементы:

  • Заголовок
  • Несколько параграфов с информацией о вас
  • Ваша фотография
  • Ссылка на ваш любимый веб-сайт или на ваш аккаунт в социальной сети
