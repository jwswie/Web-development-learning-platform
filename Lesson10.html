Расположение элементов: блочная модель, позиционирование, вложенность

Когда вы создаёте веб-сайт, важно знать, как располагать элементы на странице так, чтобы они выглядели красиво и удобно
для пользователя. Есть несколько способов расположения элементов, которые мы рассмотрим: блочная модель, позиционирование и вложенность

Блочная модель:

Основной способ расположения элементов на странице. Каждый элемент в HTML, такой как <div> или <p>, по умолчанию представляется как блок, который занимает всю доступную горизонтальную ширину и начинает новую строку

CSS

Позиционирование:

Способ размещения элементов точно на странице. Вы можете указать, где элемент должен находиться с помощью
CSS-свойства position. Оно может быть relative, absolute, fixed или sticky

HTML

CSS

Вложенность:

Это когда один элемент находится внутри другого. Вложенные элементы могут наследовать стили и находиться внутри блока родительского элемента

HTML

CSS

Правильное расположение элементов делает ваш сайт более привлекательным и удобным для пользователей. Давайте потренеруемся

Задание на закрепление материала:

Возьмите веб-страницу с блогом из HTML Урок 2. Добавьте на неё такие стили:

  • Центрируйте текст заголовка страницы
  • Задайте каждому параграфу фиксированную ширину и высоту
  • Используйте разные методы позиционирования для блоков (например, абсолютное, относительное, фиксированное)
  • Сделайте один из параграфов вложенным в другой
